+ How to Install Cypress?
  - npm install cypress --save-dev
  - npm install -D cypress
  - Add this at package.json:
      {
        "scripts": {
          "cypress:open": "cypress open"
        }
      }
  - npm run cypress:open
  - Cypress GUI will be open
  - Select "Component Test"
